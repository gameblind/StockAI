净流入额排名

API接口：http://api.biyingapi.com/higg/jlr/您的licence

备用接口：http://api1.biyingapi.com/higg/jlr/您的licence

包年版专用接口：http://n.biyingapi.com/higg/jlr/您的licence

白金版专用接口：http://b.biyingapi.com/higg/jlr/您的licence

接口说明：个股净流入额倒序排列。

数据更新：每天15:30（约10分钟完成更新）

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
t string 服务器更新时间yyyy-MM-dd HH:mm:ss
mc string 名称
dm string 代码
zxj number 最新价（元）
zdf number 涨跌幅（%）
hsl number 换手率（%）
cje number 成交额（元）
lczj number 流出资金（元）
lrzj number 流入资金（元）
jlr string 净流入（元）
jlrl string 净流入率（%）
