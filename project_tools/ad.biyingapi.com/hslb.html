沪深两市股票

API接口：http://api.biyingapi.com/hslt/list/您的licence

备用接口：http://api1.biyingapi.com/hslt/list/您的licence

包年版专用接口：http://n.biyingapi.com/hslt/list/您的licence

白金版专用接口：http://b.biyingapi.com/hslt/list/您的licence

接口说明：获取沪深两市所有股票的代码、名称、所属交易所信息，用于后续接口的传递参数。

数据更新：每日16:00更新

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
dm string 股票的六位交易代码，例如：601398
mc string 股票名称，例如：工商银行
jys string 交易所，"sh"表示上证，"sz"表示深证

沪深两市新股日历

API接口：http://api.biyingapi.com/hslt/new/您的licence

备用接口：http://api1.biyingapi.com/hslt/new/您的licence

包年版专用接口：http://n.biyingapi.com/hslt/new/您的licence

白金版专用接口：http://b.biyingapi.com/hslt/new/您的licence

接口说明：沪深两市的新股日历，按上网发行日期倒序排列

数据更新：每日16:00更新

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
zqdm string 股票代码
zqjc string 股票简称
sgdm string 申购代码
fxsl number 发行总数（股）
swfxsl number 网上发行（股）
sgsx number 申购上限（股）
dgsz number 顶格申购需配市值(元)
sgrq string 申购日期
fxjg number 发行价格（元），null为“未知”
zxj number 最新价（元），null为“未知”
srspj number 首日收盘价（元），null为“未知”
zqgbrq string 中签号公布日，null为未知
zqjkrq string 中签缴款日，null为未知
ssrq string 上市日期，null为未知
syl number 发行市盈率，null为“未知”
hysyl number 行业市盈率
wszql number 中签率（%），null为“未知”
yzbsl number 连续一字板数量，null为“未知”
zf number 涨幅（%），null为“未知”
yqhl number 每中一签获利（元），null为“未知”
zyyw string 主营业务
