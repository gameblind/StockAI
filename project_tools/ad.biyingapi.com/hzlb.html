沪深主要指数列表接口

API接口：http://api.biyingapi.com/zs/all/您的licence

备用接口：http://api1.biyingapi.com/zs/all/您的licence

包年版专用接口：http://n.biyingapi.com/zs/all/您的licence

白金版专用接口：http://b.biyingapi.com/zs/all/您的licence

接口说明：获取沪深两市主要的指数代码和名称，用于后续接口的参数传入。

数据更新：每天15:35点

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
dm string 指数代码，如：sh000001（注意，股票代码可以不带“sh/sz”前缀，指数代码必须带）
mc string 股票名称，如：上证指数
jys string 交易所，sh表示上证，sz表示深证

沪市指数列表接口

API接口：http://api.biyingapi.com/zs/sh/您的licence

备用接口：http://api1.biyingapi.com/zs/sh/您的licence

包年版专用接口：http://n.biyingapi.com/zs/sh/您的licence

白金版专用接口：http://b.biyingapi.com/zs/sh/您的licence

接口说明：获取沪市下的所有指数代码和名称，用于后续接口的参数传入。

数据更新：每天15:35点

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
dm string 指数代码，如：sh000001（注意，股票代码可以不带“sh/sz”前缀，指数代码必须带）
mc string 股票名称，如：上证指数
jys string 交易所，sh表示上证，sz表示深证（因为此接口只有沪市指数，所以参数返回只有sh）
