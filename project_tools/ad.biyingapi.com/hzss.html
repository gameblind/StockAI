指数实时数据接口

API接口：http://api.biyingapi.com/zs/sssj/指数代码(如sh000001)/您的licence

备用接口：http://api1.biyingapi.com/zs/sssj/指数代码(如sh000001)/您的licence

包年版专用接口：http://n.biyingapi.com/zs/sssj/指数代码(如sh000001)/您的licence

白金版专用接口：http://b.biyingapi.com/zs/sssj/指数代码(如sh000001)/您的licence

接口说明：根据《指数列表》得到的指数代码（包含sh/sz前缀，如：sh000001）获取实时交易数据，您可以理解为日线的最新数据。

数据更新：交易时间段每1分钟

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
h number 最高
l number 最低
o number 开盘
pc number 涨跌幅（%）
p number 当前指数值
cje number 成交额（元）
ud number 涨跌额
v number 成交量（手）
yc number 昨日收盘
t string 更新时间yyyy-MM-dd HH:mm:ss

沪深两市上涨下跌数概览

API接口：http://api.biyingapi.com/zs/lsgl/您的licence

备用接口：http://api1.biyingapi.com/zs/lsgl/您的licence

包年版专用接口：http://n.biyingapi.com/zs/lsgl/您的licence

白金版专用接口：http://b.biyingapi.com/zs/lsgl/您的licence

接口说明：获取沪深两市的股票上涨、下跌总数以及各比例区间的上涨下跌数。

数据更新：交易时间段每2分钟

请求频率：1分钟20次 | 包年版1分钟3000次 | 白金版1分钟6000次

返回格式：标准Json格式 [{},...{}]

字段名称 数据类型 字段说明
totalUp number 上涨总数
totalDown number 下跌总数
zt number 涨停总数
dt number 跌停总数
up8ToZt number 上涨8%~涨停的数量
up6To8 number 上涨6%~8%的数量
up4To6 number 上涨4%~6%的数量
up2To4 number 上涨2%~4%的数量
up0To2 number 上涨0%~2%的数量
down0To2 number 下跌0%~-2%的数量
down2To4 number 下跌-2%~-4%的数量
down4To6 number 下跌-4%~-6%的数量
down6To8 number 下跌-6%~-8%的数量
down8ToDt number 下跌-8%~跌停的数量
