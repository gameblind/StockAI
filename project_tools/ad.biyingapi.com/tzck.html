白金版专用接口：http://b.biyingapi.com/hitc/jrts/您的licence

接口说明：今日股票、基金公告事项以及交易异动概览。

数据更新：每天15:30（约10分钟更新完成）

jjfp string 基金收益分配款发放日。格式：jjfp:[xxx,xxx,...]。其中xxx格式为“(股票代码) 股票名称：说明”
jjcx string 基金收益分配除息日。格式：jjcx:[xxx,xxx,...]。其中xxx格式为“(股票代码) 股票名称：说明”

szgg string 深交所公告。格式：szgg:[xxx,xxx,...]。其中xxx格式为“(股票代码) 股票名称：说明”
